# README
Description
This project is a personal "About Me" website used to learn more about  HTML CSS. We learned about  structure, styling, and responsive design. This was also a way of being expressive  in regards to technology. Personally, I enjoyed this project and this class. We started with basic concepts and adding  preliminary elements like header body h1 and using different tools to make avatars  our ourselves for this project. We were given a great deal of artistic control so a long the way we had to make decision about the style and structure that got us to this point. This project taught me how to use HTML elements, tags, and attributes to structure a webpage in ways I didn’t  think of in the past. We learned  about media queries  flexes the rule of specificity floats etc. We also learned about making web pages that are considerate for impaired users, the box model and much more  . We also were given the opportunity to take creative control and be virtuosic. For instance I started with a green scheme but after working with my professor I landed on warm  colors like E38417, E3D017, d37315. These colors were a great decision because I do prefer this look. I feel like it represents myself better.  I also added font families rye and olea script. This look was fitting for me and I enjoy  what it added to my  website. This page is the product of what I have learned over the last few weeks and mixed with a bit of who I am. I hope you enjoy it!
